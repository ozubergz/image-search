API IMAGE SEARCH

Description

This is an API service that can search images that are related to a given search word which you provide. Search results output is in JSON format.

Example API calls

https://img-abstract.glitch.me/api/search/cat?offset=6

serve routes: search/(:word) and ?offset=(number)

Offset number is the page of the results. Providing offset number is optional. If offset is not provided, default value is 1.
